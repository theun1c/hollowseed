# feat-002 простыми словами

## Что делаем
На этом шаге мы делим всю карту на несколько прямоугольных секций.

## Зачем это нужно
Секции станут "контейнерами" для будущих комнат в следующих фичах.

## Что важно сейчас
- Никаких комнат пока не создаем
- Никаких коридоров пока не делаем
- Проверяем только корректность разбиения (границы и размеры)

## Результат фичи
Функция разбиения возвращает валидный список секций, который можно использовать дальше.
